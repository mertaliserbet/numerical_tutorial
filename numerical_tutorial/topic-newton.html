Newton-Raphson Method and Comparison with Bisection

1. What is the Newton-Raphson Method?

The Newton-Raphson method is a fast numerical technique used to find the root of a function by utilizing its derivative. The core idea is to use the intersection point of the tangent line with the x-axis as the next approximation.

2. Mathematical Definition and Formula

The derivative-based approach for a function at point \( x_0 \):

$$ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} $$

  • \( f(x) \): the function
  • \( f'(x) \): the derivative of the function
  • \( x_n \): current approximation
  • \( x_{n+1} \): next approximation

3. Algorithm Steps

  1. Choose an initial guess: \( x_0 \)
  2. Iterate using: \( x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} \)
  3. Stop when \( \left| x_{n+1} - x_n \right| < \varepsilon \)

4. Advantages ‚úÖ

  • Quadratic convergence: Very fast results with a good initial guess.
  • High accuracy can be achieved.

5. Disadvantages ‚ùå

  • Requires derivative: You must know or compute the derivative.
  • May diverge if the initial guess is far from the root.
  • Division error if \( f'(x) = 0 \).

6. Python Example

def f(x):
    return x**3 - x - 2

def df(x):
    return 3*x**2 - 1

def newton_raphson(x0, tol=1e-6, max_iter=100):
    for _ in range(max_iter):
        fx = f(x0)
        dfx = df(x0)
        if dfx == 0:
            raise ZeroDivisionError("f'(x) = 0")
        x1 = x0 - fx / dfx
        if abs(x1 - x0) < tol:
            return x1
        x0 = x1
    return x0
root = newton_raphson(1.5) print(f"Approximate root: {root}")

7. Newton-Raphson vs. Bisection Comparison

Feature Bisection Method Newton-Raphson Method
Convergence Guaranteed if [a, b] brackets a root and f is continuous. Not guaranteed; may diverge.
Convergence Rate Linear convergence. Quadratic convergence.
Initial Input Requires interval [a, b] where f(a) and f(b) have opposite signs. Requires a single initial guess x‚ÇÄ.
Function Requirement Function must be continuous. Function must be differentiable.
Robustness Very robust with a good interval. Less robust, sensitive to initial guess.
Iteration Cost Low: midpoint and function evaluations. Higher: includes derivative computations.
Use Case When convergence is critical and derivative is unknown. When speed is important and derivative is known.

8. When to Use Which Method?

Bisection Method:
  • When guaranteed convergence is needed
  • When no good initial guess is available
  • When the function is not differentiable or behaves poorly
  • As a fallback method
Newton-Raphson Method:
  • When faster convergence is desired
  • When a good initial guess is available
  • When the function is well-behaved and differentiable
  • Used in optimization and advanced numerical tasks

9. Conclusion

There is no universally "best" method; it depends on the problem:

  • Bisection: Slower but always reliable.
  • Newton-Raphson: Faster but more sensitive and requires care.

In practice, a hybrid strategy is often ideal: first narrow the root with Bisection, then refine with Newton-Raphson.
