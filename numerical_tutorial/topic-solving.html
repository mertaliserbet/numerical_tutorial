Solving Many Equations (Linear Systems)

This page provides a detailed educational overview of solving linear equation systems in numerical methods, including core concepts, practical applications, and implementation using Python/NumPy. You'll understand how critical these systems are in engineering, science, economics, and computer science.

1. Introduction to Linear Equation Systems

When multiple unknowns (x, y, z, ...) are involved, they are expected to simultaneously satisfy certain conditions. These situations are expressed through multiple equations, forming what is known as a "system of linear equations."

Main Objective: Find the values of the unknowns that satisfy all equations at the same time.

Simple Example:

  • 2x + y = 4
  • x - y = -1

Solving this system means finding the values of x and y that satisfy both equations simultaneously. These are called the solution of the system.

2. Real-Life Applications

  • Electric Circuits: According to Kirchhoff's laws...
  • Resource Allocation: Product distribution based on limited materials.
  • Structural Analysis: Force equilibrium equations in bridges, etc.
  • Economics: Equilibrium pricing via supply-demand.
  • Data Fitting: Line or curve fitting for datasets.

3. What Does "Linear" Mean?

  • All variables appear to the first power only (no x¬≤, sin(x), etc.).
  • Variables do not multiply each other.
  • Variables combined via + / ‚àí and multiplied by constants.

Linear: 3x + 2y = 5 ‚úÖ
Nonlinear: x¬≤ + y = 5 ‚ùå, sin(x) + y = 1 ‚ùå, x¬∑y = 3 ‚ùå

4. Standard Form: Ax = b

A * x = b
  • A: Coefficient matrix
  • x: Unknown vector
  • b: Constant vector

5. Representing in Python (NumPy)

import numpy as np
A = np.array([[2, 1], [1, -1]])
b = np.array([4, -1])

6. Solving with numpy.linalg.solve()

x = np.linalg.solve(A, b)
print(x)

7. Singular and Non-Singular Systems

try:
  x = np.linalg.solve(A, b)
except np.linalg.LinAlgError:
  print("Matrix is singular!")

8. Geometric Interpretation

  • Intersecting: Unique solution ‚úÖ

12. Practical Code Exercises

A. Solving a 3x3 System
A = np.array([[1, 1, 2], [2, 4, -3], [3, 6, -5]])

B. Verifying Solutions
np.allclose(A @ x, b)
error = np.linalg.norm(b - A @ x)
