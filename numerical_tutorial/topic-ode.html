1. Ordinary Differential Equations (ODEs)

An Ordinary Differential Equation (ODE) is an equation that contains derivatives of an unknown function. It is typically written in the form:

\[ \frac{dy}{dx} = f(x, y) \]

Where:

  • y is the unknown function,
  • x is the independent variable,
  • f(x, y) defines the derivative dy/dx.

2. What is Our Goal?

Solving an ODE numerically means finding approximate values of y starting from a known initial value, step by step.

Initial Value Problem (IVP):

\[ \frac{dy}{dx} = f(x, y), \quad y(x_0) = y_0 \]

The goal is to approximate y at successive points like x‚ÇÅ = x‚ÇÄ + h, x‚ÇÇ = x‚ÇÄ + 2h, etc.

3. Euler‚Äôs Method

This is the simplest numerical method. The core idea is:

\[ y_{n+1} = y_n + h \cdot f(x_n, y_n) \]

Where:

  • h = step size
  • (x‚Çô, y‚Çô) = current point
  • f(x‚Çô, y‚Çô) = slope (i.e., the derivative)

Steps:

  1. Start with the initial point \( (x_0, y_0) \)
  2. Compute the next value: \( y_1 = y_0 + h \cdot f(x_0, y_0) \)
  3. Continue: \( y_2 = y_1 + h \cdot f(x_1, y_1) \), and so on.

Pros: Simple and fast
Cons: Low accuracy, especially with larger h

4. Runge-Kutta Methods

These are more advanced and accurate methods than Euler‚Äôs. The most common is the 4th Order Runge-Kutta (RK4) method.

RK4 Formula:

\[ \begin{aligned} k_1 &= f(x_n, y_n) \\ k_2 &= f\left(x_n + \frac{h}{2}, y_n + \frac{h}{2}k_1\right) \\ k_3 &= f\left(x_n + \frac{h}{2}, y_n + \frac{h}{2}k_2\right) \\ k_4 &= f(x_n + h, y_n + h k_3) \\ y_{n+1} &= y_n + \frac{h}{6} (k_1 + 2k_2 + 2k_3 + k_4) \end{aligned} \]

It gives a much better approximation using four slope estimates per step.

5. Example Problem

Given:

\[ \frac{dy}{dx} = x + y, \quad y(0) = 1, \quad h = 0.1 \]

Euler Step:

\[ y_1 = y_0 + h \cdot f(x_0, y_0) = 1 + 0.1 \cdot (0 + 1) = 1.1 \]

Runge-Kutta Step:

\[ \begin{aligned} k_1 &= f(0, 1) = 1 \\ k_2 &= f(0.05, 1.05) = 1.1 \\ k_3 &= f(0.05, 1.055) = 1.105 \\ k_4 &= f(0.1, 1.1105) = 1.2105 \\ y_1 &= 1 + \frac{0.1}{6}(1 + 2(1.1) + 2(1.105) + 1.2105) \approx 1.1103 \end{aligned} \]

Euler gives a rough result of 1.1, while RK4 yields a more accurate 1.1103.

7. Error Analysis

  • Euler Method: O(h) error
  • RK4 Method: O(h‚Å¥) error

So RK4 gives significantly better accuracy with the same step size, though it requires more computation.

9. Python Implementation (Simple Example)

def f(x, y):
    return x + y

def euler(x0, y0, h, steps):
    for i in range(steps):
        y0 += h * f(x0, y0)
        x0 += h
        print(f"x={x0:.2f}, y={y0:.4f}")

def rk4(x0, y0, h, steps):
    for i in range(steps):
        k1 = f(x0, y0)
        k2 = f(x0 + h/2, y0 + h*k1/2)
        k3 = f(x0 + h/2, y0 + h*k2/2)
        k4 = f(x0 + h, y0 + h*k3)
        y0 += h/6 * (k1 + 2*k2 + 2*k3 + k4)
        x0 += h
        print(f"x={x0:.2f}, y={y0:.4f}")

10. Summary

  • ODEs are fundamental in modeling physical systems.
  • Numerical methods let us solve \( \frac{dy}{dx} = f(x,y) \) problems step-by-step.
  • Euler‚Äôs method is simple but inaccurate.
  • Runge-Kutta (especially 4th order) is widely used for better accuracy.
  • ODE solvers are used in physics, biology, engineering, and more.
