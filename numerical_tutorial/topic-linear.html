Linear and Polynomial Interpolation

1. What is Interpolation?

Interpolation is a technique used to estimate unknown values between known data points. It is especially useful in experimental data or functions with limited measurement points to "fill in the gaps."

Real-world examples:

  • Estimating missing sensor readings between known time points
  • Filling in missing days in financial time series
  • Creating smooth movements in computer graphics

2. Linear Interpolation

The simplest interpolation method. It assumes a straight-line relationship between two known data points.

Formula:

$$y = y_0 + \frac{(x - x_0)(y_1 - y_0)}{x_1 - x_0}$$

Where:

  • \((x_0, y_0)\) and \((x_1, y_1)\): known data points
  • \(x\): intermediate value where we want to estimate \(y\)
  • \(y\): estimated value

Example:

Given: (2, 4) and (6, 8). Find: y when x = 4
$$y = 4 + \frac{(4 - 2)(8 - 4)}{6 - 2} = 4 + \frac{2 \cdot 4}{4} = 4 + 2 = 6$$

3. Polynomial Interpolation

Used when a single polynomial is required to pass through all data points. The general form:

$$P(x) = a_0 + a_1 x + a_2 x^2 + \dots + a_n x^n$$

If there are \(n+1\) points, a polynomial of degree \(n\) can pass through all of them. But note:

  • Higher degrees can lead to oscillation.
  • Runge's phenomenon may occur, causing instability.

4. Lagrange Interpolation

One of the most commonly used polynomial methods. Formula:

$$L(x) = \sum_{j=0}^{n} y_j \cdot \ell_j(x)$$
Where:
$$\ell_j(x) = \prod_{\substack{0 \leq m \leq n \\ m \ne j}} \frac{x - x_m}{x_j - x_m}$$

Each term contributes based on the influence of a single \(y_j\). Easy to code, but computationally intensive for large datasets.

5. Newton Interpolation

An alternative method based on Newton's forward differences.

$$P(x) = a_0 + a_1(x - x_0) + a_2(x - x_0)(x - x_1) + \dots$$

Advantage: It can be expanded step-by-step as new data is added. Ideal for dynamic datasets.

6. Cubic Spline Interpolation

Instead of a single polynomial, a cubic polynomial is fit between each interval. Goal: smoother and more accurate transitions.

Spline functions satisfy:

  • The function is continuous across all intervals
  • Derivatives are also continuous
  • It passes through all data points

Used widely in graphics rendering, animations, and CAD applications.

7. Comparison Table

Method Advantages Disadvantages Application Areas
Linear Fast, simple Low accuracy, sharp transitions Basic predictions, simple sensor data
Lagrange Theoretically strong Heavy computation on large datasets Mathematical analysis
Newton Expandable step-by-step Similar to Lagrange Data streams, dynamic updates
Cubic Spline Smooth and visually consistent More complex to implement Graphics, physics simulation, animation

8. Conclusion

Interpolation is a fundamental tool for making meaningful estimates from limited data. Choosing the right method is critical for both computational efficiency and result accuracy.

  • For simplicity and speed: Linear
  • For theoretical accuracy: Lagrange/Newton
  • For realism and visual smoothness: Cubic Spline

Each method becomes a powerful mathematical tool when applied in the right context.
