Bisection Method

1. What Is the Bisection Method?

The Bisection Method is one of the simplest and most reliable numerical methods used for solving root-finding problems. It is based on the Intermediate Value Theorem, which states:

If a continuous function \( f(x) \) has opposite signs at the endpoints of an interval \([a, b]\), i.e.:
\( f(a) \cdot f(b) < 0 \)
then there exists at least one root \( c \in (a, b) \) such that \( f(c) = 0 \).

2. Why Use Bisection?

In many real-world applications, we encounter equations that:

  • Have no analytical solutions
  • Involve transcendental functions (e.g., \( \sin(x) \), \( e^x \))
  • Are too complex or nonlinear for algebraic solving

In such cases, the Bisection Method gives us a guaranteed and simple way to approximate a root.

3. Step-by-Step Algorithm

  1. Compute the midpoint: \( c = \frac{a + b}{2} \)
  2. Evaluate \( f(c) \)
  3. Check the sign:
    • If \( f(a) \cdot f(c) < 0 \), set \( b=c \)
    • If \( f(c) \cdot f(b) < 0 \), set \( a=c \)
  4. Repeat until \( |b - a| < \varepsilon \) or \( |f(c)| < \varepsilon \)

Each iteration cuts the interval in half, bringing us closer to the root.

4. Mathematical Analysis

  • Interval length after \( n \) steps: $$ |b_n - a_n| = \frac{1}{2^n}(b_0 - a_0) $$
  • To reach error \( \varepsilon \): $$ n \ge \log_2\left(\frac{b_0 - a_0}{\varepsilon}\right) $$

So the convergence is linear.

5. Advantages ‚úÖ

  • Guaranteed to converge if \( f(a) \cdot f(b) < 0 \)
  • No derivative needed
  • Simple to implement
  • Numerically stable

6. Disadvantages ‚ùå

  • Slow convergence (linear rate)
  • Requires sign change in the interval
  • Only one root is found per interval

7. Python Example

def f(x):
    return x**3 - x - 2

def bisection(a, b, tol=1e-6, max_iter=100):
    if f(a) * f(b) >= 0:
        raise ValueError("f(a) and f(b) must have opposite signs.")

    for i in range(max_iter):
        c = (a + b) / 2.0
        if abs(f(c)) < tol or abs(b - a) < tol:
            return c
        if f(a) * f(c) < 0:
            b = c
        else:
            a = c
    return (a + b) / 2.0

root = bisection(1, 2)
print(f"Approximate root: {root}")

This solves \( x^3 - x - 2 = 0 \), whose root is approximately 1.521...

9. Final Note

The Bisection Method is a foundational numerical technique. It is not the fastest, but it is the most reliable. It's ideal for teaching, testing, and use in situations where robustness is more important than speed.
