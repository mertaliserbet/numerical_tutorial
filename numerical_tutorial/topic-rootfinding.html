Solving Equations and Root-Finding Methods

1. Why Is Numerical Root Finding Necessary?

In many real-world problems, we need to find the zero (root) of a function, where \( f(x) = 0 \). However, it is not always possible to find these roots using algebraic methods. Especially in cases involving:

  • Equations without analytical solutions,
  • Transcendental functions (e.g., \( \sin(x) \), \( e^x \)),
  • Complex or large systems...

In such situations, numerical methods are needed to find approximate solutions. Numerical root-finding allows us to determine the roots of such functions accurately using computational techniques.

2. Basic Concept: What Is a Root?

A root of a function is a value of \( x \) such that \( f(x) = 0 \) — in other words, where the graph of the function crosses the x-axis.

  • If \( f(x) > 0 \), the graph lies above the x-axis.
  • If \( f(x) < 0 \), it lies below.
  • If \( f(x) = 0 \), the graph intersects the x-axis: this point is the root.

3. Bisection Method

Precondition: \( f(a) \cdot f(b) < 0 \), meaning the function changes sign in the interval \([a, b]\).

Steps:

  1. Calculate the midpoint: \( c = \frac{a + b}{2} \)
  2. Check the sign of \( f(c) \):
    • If \( f(a) \cdot f(c) < 0 \) → update to \([a, c]\)
    • If \( f(c) \cdot f(b) < 0 \) → update to \([c, b]\)
  3. Repeat until the interval is below a desired tolerance.

Advantages: Simple, always converges (if a root exists).
Disadvantages: Converges slowly (linearly), needs a valid initial interval.

4. Newton-Raphson Method

Formula: $$ x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} $$

  1. Choose initial guess \( x_0 \)
  2. Update using the formula
  3. Repeat until \( \left| x_{n+1} - x_n \right| < \text{tolerance} \)

Advantages: Very fast (quadratic convergence)
Disadvantages: Requires derivative, may diverge, division by zero if \( f'(x) = 0 \)

5. Secant Method

Formula: $$ x_{n+1} = x_n - f(x_n) \cdot \frac{x_n - x_{n-1}}{f(x_n) - f(x_{n-1})} $$

Advantages: Derivative-free, converges nearly as fast as Newton-Raphson.
Disadvantages: Needs two initial values, can be unstable.

6. Comparison Table

Method Requires Derivative Convergence Stability Notes
Bisection No Linear Very High Guaranteed convergence
Newton-Raphson Yes Quadratic Low to Medium Sensitive to derivative
Secant No Near Quadratic Medium No derivative, but sensitive

7. Real-World Examples

  • Economics: Finding break-even points
  • Physics: Finding when velocity = 0
  • Electrical Engineering: Voltage balance
  • Mechanical Engineering: Equilibrium of systems

8. Conclusion

Root-finding methods are fundamental in numerical analysis. Each method has advantages and drawbacks. The key is selecting the most suitable approach based on the problem structure. These techniques power the core of many computational tools.
