Least Squares and Curve Fitting

1. Introduction: Why Curve Fitting?

The Problem: We are given many data points, but they do not exactly follow a known mathematical function. In this case, we cannot use interpolation, because interpolation assumes the function must pass through all given points exactly.

The Solution: Instead, we search for a function that approximately follows the trend of the data. This process is called Curve Fitting, and the most common technique is the Least Squares Method.

2. Problem Definition

Given data points:

\( (x_1, y_1), (x_2, y_2), \dots, (x_n, y_n) \)

We aim to find a function \( f(x) \) that fits the data "as well as possible." Typically, this function is a line, parabola, or polynomial.

3. The Main Idea of the Least Squares Method

The goal is to minimize the total squared error between the actual \( y_i \) values and the predicted \( f(x_i) \) values:

\[ S = \sum_{i=1}^{n} [y_i - f(x_i)]^2 \]

This function \( S \) is the error function, and we aim to minimize it by choosing the best parameters of \( f(x) \).

4. Simplest Case: Linear Regression

We assume the function is a straight line:

\( f(x) = ax + b \)

We aim to find the best values of \( a \) and \( b \) such that this line fits the data well.

4.1. Goal:

Minimize the squared error:

\[ S = \sum_{i=1}^{n} [y_i - (ax_i + b)]^2 \]

We take the partial derivatives of \( S \) with respect to \( a \) and \( b \), set them to zero, and solve — this gives us the normal equations.

5. Normal Equations (for Linear Fit)

We obtain a system of two equations:

\[ \begin{cases} a \sum x_i^2 + b \sum x_i = \sum x_i y_i \\ a \sum x_i + b n = \sum y_i \end{cases} \]

Solving this system gives us the best-fit values of \( a \) and \( b \), and the resulting best-fit line is: \( y = ax + b \)

6. Applications

  • Data Science: Trend analysis in datasets
  • Machine Learning: Core of regression models
  • Engineering: Handling measurement errors
  • Physics: Deriving experimental formulas from noisy data

7. Polynomial Curve Fitting (Higher-Degree)

We are not limited to lines; we can fit polynomials like:

\( f(x) = a_2 x^2 + a_1 x + a_0 \)

The same least squares principle applies. We construct and solve the normal equations for all coefficients \( a_i \).

This can also be solved in matrix form:

\( A^T A \vec{a} = A^T \vec{y} \)

Where:

  • \( A \): Design matrix (based on input \( x \))
  • \( \vec{a} \): Coefficient vector
  • \( \vec{y} \): Output values vector

8. Interpolation vs. Curve Fitting

Feature Interpolation Curve Fitting
Number of points Few Many
Fits all points? Yes (exactly) No (approximately)
Purpose Pass through all points Best approximate function
Examples Lagrange, Newton Least Squares Method

9. Python/NumPy Example (Optional)

import numpy as np
import matplotlib.pyplot as plt

x = np.array([1, 2, 3, 4, 5])
y = np.array([2.2, 2.8, 3.6, 4.5, 5.1])

# Linear fit (degree 1)
a, b = np.polyfit(x, y, 1)

# Predicted values
y_fit = a*x + b

plt.scatter(x, y, color='red', label='Data')

plt.legend()
plt.show()
